Voters' Vote

YOU WILL NEED AN ETHEREUM WALLET TO RUN THIS DAPP

This is a Voting app which is backed by blockchain and records each vote made by a voter without double voting.
The voters are given Unique address to cast votes After that, users use their ethereum wallet account address to submit votes, during which the application checks to see if the user with this address has voted before.
If this particular address has not been used before, the vote goes through the contract and the vote count is updated
